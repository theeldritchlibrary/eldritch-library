The files in this directory are for use with the Tidbits module. This module automatically shows helpful tips to your players when they join the game, during load screens and when the game is paused. 


Tidbits
An Add-on Module by TheRipper93
URL: https://foundryvtt.com/packages/tidbits

Files
- tidbits-bgs\shadowdark-tidbits-bg.webp - an image file used by the module during loading scenes. 
- shadowdark-tidbits.txt - a text file with dozens of different shadowdark tips and lore

To Use
1. You need to load and activate the Tidbits modules
2. Go into Game Settings > Configure Settings > Tidbits
3. Check (select) teh Pause text checkbox
4. Uncheck (deselect) the Use default sources checkbox
5. Change the Custom source to point to the shadowdark-tidbits.txt file
6. Change the Image source to point to tidbits-bgs folder
7. Save Changes

Now, when you load scenes or use the Pause feature, you'll see the shadowdark tidbits!
